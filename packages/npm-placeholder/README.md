# aipermission

This is a placeholder package for the AIPermission project.

Use the official scoped MCP bridge package instead:

```bash
npx -y @aipermission/mcp
```

Project docs:

https://github.com/aipermission/aipermission

Demo video:

https://github.com/aipermission/aipermission/releases/download/v0.1.0-rc.1/aipermission-demo.mp4

The real gateway runs locally through Docker Compose. The npm MCP package only connects AI clients to that local gateway.
